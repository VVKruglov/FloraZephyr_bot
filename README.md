📱 О боте
Telegram-бот для магазина зефирных композиций "ФлораЗефир". Бот предоставляет каталог товаров, информацию об акциях, составе продукции и способах доставки.
🛠 Технологии
Python 3.7+
python-telegram-bot (версия 20.x)
Работа с Inline-клавиатурами и Reply-клавиатурами
🎮 Команды бота
Команда	Описание
/start	Начало работы, приветственное сообщение
/catalog	Показать каталог товаров
/promotion	Показать текущие акции
🖥 Кнопки меню
Кнопка	Действие
Каталог	Открыть каталог товаров
Акции	Показать текущие акции
Состав	Информация о составе продукции
Доставка	Информация о способах доставки
