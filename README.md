*ФлораЗефир — Telegram-бот*

*О боте*
Бот для магазина зефирных композиций "ФлораЗефир". Предоставляет каталог товаров, информацию об акциях, составе и доставке.

*Технологии*
Python 3.7+ | python-telegram-bot 20.x | Inline/Reply-клавиатуры

*Команды*
/start — Приветствие
/catalog — Каталог товаров
/promotion — Текущие акции

*Кнопки меню*
Каталог — Открыть каталог
Акции — Показать акции
Состав — Состав продукции
Доставка — Способы доставки
